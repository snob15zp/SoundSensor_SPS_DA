General Description for the DA14585-531 Serial Port Service Application

Serial Port Service

The Serial Port Service (SPS) emulates a serial cable communication. It provides a simple substitute for RS-232 connections, including the familiar software flow control logic via BluetoothÂ® Low Energy. The SPS software distribution includes the application and profile source codes. There are two projects provided, host and device projects, operating in BLE central and epripheral role respectively. Both projects use DMA driven UART operation for baud rates up to 921600, using hardware flow control only, optimized for DLE operation and high speeds. Both host and device applications are fully configurable, by reading a preprogrammed configuration file in SPI flash, while device application supports a remote configuration service server and it can be controlled by a connected smartphone app in runtime.
