86 /*
87  ****************************************************************************************
88  *
89  * Advertising or scan response data for the following cases:
90  *
91  * - ADV_IND: Connectable undirected advertising event.
92  * - The maximum length of the user defined advertising data shall be 28 bytes.
93  * - The Flags data type are written by the related ROM function, hence the user shall
94  * not include them in the advertising data. The related ROM function adds 3 bytes in
95  * the start of the advertising data that are to be transmitted over the air.
96  * - The maximum length of the user defined response data shall be 31 bytes.
97  *
98  * - ADV_NONCONN_IND: Non-connectable undirected advertising event.
99  * - The maximum length of the user defined advertising data shall be 31 bytes.
100  * - The Flags data type may be omitted, hence the user can use all the 31 bytes for
101  * data.
102  * - The scan response data shall be empty.
103  *
104  * - ADV_SCAN_IND: Scannable undirected advertising event.
105  * - The maximum length of the user defined advertising data shall be 31 bytes.
106  * - The Flags data type may be omitted, hence the user can use all the 31 bytes for
107  * data.
108  * - The maximum length of the user defined response data shall be 31 bytes.
109  ****************************************************************************************
110  */
