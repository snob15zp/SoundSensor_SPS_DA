# SoundSensor_SPS_DA
project file in \projects\target_apps\dsps\dsps_device\Keil_5


Три варианта сборки:
Dsps_585 – этот вариант из примера, без изменений
Dsps_531 – этот вариант из примера тоже, но модифицирован под USB KIT
SoundSensor_Fix – этот вариант добавлен под разрабатываемое устройство.

Варианты помимо состава файлов проекта, отличаются define в настройках проекта:

1. Dsps_531: здесь нужен только define __DA14531__
   2. SoundSensor_Fix: здесь нужны макроопределения 
       А)__DA14531__ чтобы библиотеки переключились на da14531
Б) __SoundSensor__ , это определение говорит о том, что это не USB KIT, то есть некоторый код только для usb kit отключается, код только для DUD включается.
В) __SSFix__ разрешает код предназначенный для перепаянных плат. Сейчас все платы перепаяны. Этот код относится к распределению ног SPI
Г) __ADCTEST__ включает код для получения сырых данных ADC. Все остальные функции при этом не работают не работают
       Е) __BOARD2__ : не отключает LDO процессора. Это необходимо для платы 2, так как на этой плате не работает DCDC.
