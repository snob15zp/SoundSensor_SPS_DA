81 /*
82  ****************************************************************************************
83  *
84  * Advertising or scan response data for the following cases:
85  *
86  * - ADV_IND: Connectable undirected advertising event.
87  * - The maximum length of the user defined advertising data shall be 28 bytes.
88  * - The Flags data type are written by the related ROM function, hence the user shall
89  * not include them in the advertising data. The related ROM function adds 3 bytes in
90  * the start of the advertising data that are to be transmitted over the air.
91  * - The maximum length of the user defined response data shall be 31 bytes.
92  *
93  * - ADV_NONCONN_IND: Non-connectable undirected advertising event.
94  * - The maximum length of the user defined advertising data shall be 31 bytes.
95  * - The Flags data type may be omitted, hence the user can use all the 31 bytes for
96  * data.
97  * - The scan response data shall be empty.
98  *
99  * - ADV_SCAN_IND: Scannable undirected advertising event.
100  * - The maximum length of the user defined advertising data shall be 31 bytes.
101  * - The Flags data type may be omitted, hence the user can use all the 31 bytes for
102  * data.
103  * - The maximum length of the user defined response data shall be 31 bytes.
104  ****************************************************************************************
105  */
