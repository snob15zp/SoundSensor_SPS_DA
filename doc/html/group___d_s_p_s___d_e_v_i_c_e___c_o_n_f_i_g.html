#define APP_DIS_PNP_ID   ("\x01\xD2\x00\x80\x05\x00\x01")

PNP ID Value - LSB -> MSB Vendor ID Source : 0x02 (USB Implementers Forum assigned Vendor ID value) Vendor ID : 0x045E (Microsoft Corp) Product ID : 0x0040 Product Version : 0x0300 e.g. define APP_DIS_PNP_ID ("\x02\x5E\x04\x40\x00\x00\x03")
